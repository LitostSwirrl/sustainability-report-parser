# Scope 3 欄位 (GHG Protocol Categories) — 欄位 16-30 (V2)

> Source: `src/field_definitions.py` → `SCOPE3_FIELDS` (original IDs 42-56, mapped to V2 IDs 16-30 via `FIELD_MAPPING_V1_TO_V2`)
> Applicable to: All companies. These fields capture the 15 GHG Protocol Scope 3 categories for upstream and downstream indirect emissions. In V1 the dictionary keys are 42-56; in V2 (2026 驗證指標) they are renumbered 16-30 within `BASE_FIELDS_V2`.
